# 2025-06-21 Web 首页改造为房产网站

## 改动内容

### 后端新增
- `LoupanPublicController.java` - 公开 API 控制器（无需登录）
  - `GET /api/public/loupans` - 楼盘列表（分页+搜索+行政/板块/类型/装修筛选）
  - `GET /api/public/loupans/{id}` - 楼盘详情
  - `GET /api/public/loupans/{id}/huxings` - 户型列表
  - `GET /api/public/loupans/{id}/medias` - 媒体素材列表
  - `GET /api/public/loupan-filters` - 筛选选项（行政区/板块列表）

### Web 前端改造

| 文件 | 改动 |
|------|------|
| `views/Home.vue` | 重写为房产首页：Hero 搜索区 + 筛选栏（行政区/板块/类型/装修）+ 楼盘卡片网格 + 分页 |
| `views/LoupanDetail.vue` | **新增**：楼盘详情页，含基本信息/建筑指标/开发信息/车位/外立面/配套，户型 Tab，图库 Tab |
| `router/index.js` | 新增 `/loupan/:id` 路由（不需登录） |
| `layouts/WebLayout.vue` | 导航改为「首页」「楼盘」，底部改为「热门区域」「找房工具」 |
| `index.html` | title 改为「fang123 - 找到你的理想家园」 |
