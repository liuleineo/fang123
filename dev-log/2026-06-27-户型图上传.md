# 2026-06-27 户型管理新增户型图上传

## 需求
新建/编辑户型时可直接上传户型图片。

## 改动

### 后端
| 文件 | 改动 |
|------|------|
| `entity/LoupanHuxing.java` | 新增 `huxingImage` 字段（String，户型图URL） |

### 数据库
```sql
ALTER TABLE loupan_huxing ADD COLUMN huxing_image VARCHAR(500) DEFAULT NULL COMMENT '户型图URL';
```

### 前端 admin/src/views/LoupanHuxingManage.vue
- 表单新增"户型图"上传区（`t-upload`，单图，image/*）
- 支持上传后自动填入 URL，也可手动输入
- 复用现有 COS 上传接口 `/api/admin/medias/upload`
- `openCreate`/`openEdit` 时清空上传列表
