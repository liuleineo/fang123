# 2026-06-29 AI新建户型功能

## 概述
参照 AI 新增楼盘，在户型管理页面新增"AI新建户型"按钮，上传户型资料图片后自动识别多个户型并批量创建。

## 技术方案
```
图片 → OCR文字提取 → TokenHub解析JSON数组 → 勾选 → 批量创建
```

## 后端新增
| 文件 | 说明 |
|------|------|
| `dto/AiParseHuxingResult.java` | 户型解析结果DTO，含 `List<HuxingFields> huxings` |
| `service/AiParseService.java` | 新增 `parseHuxings()` 方法 + 户型专用提示词 |
| `controller/LoupanHuxingController.java` | 新增 `POST /api/admin/huxings/ai-parse` |

## 前端更新
- "新建户型"旁新增"AI新建户型"按钮
- AI对话框：楼盘ID输入 + 图片上传 + OCR原文折叠 + 户型列表勾选 + 批量创建
