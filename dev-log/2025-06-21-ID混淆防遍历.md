# 2025-06-21 楼盘 ID 混淆防遍历

## 问题
公开页面 URL 暴露自增 ID（如 `/loupan/1`），可被遍历查询所有数据。

## 方案
XOR 混淆 + Base64 URL-safe 编码，将数字 ID 转换为不可预测的短字符串（如 `/loupan/AAAAAAAAAQM`），不引入外部依赖。

## 修改文件

| 文件 | 改动 |
|------|------|
| `common/IdObfuscator.java` | **新增**：`encode(Long)` / `decode(String)` 方法，XOR + Base64 URL-safe |
| `entity/Loupan.java` | 新增 `@TableField(exist = false) String encodedId` 瞬态字段 |
| `controller/LoupanPublicController.java` | list 接口返回 encodedId；detail/huxing/media 接口接收 encodedId 自动解码 |
| `views/Home.vue` | 卡片链接从 `/loupan/${lp.id}` 改为 `/loupan/${lp.encodedId}` |

## 效果
- URL 从 `/loupan/1` 变为 `/loupan/AAAAAAAAAQM`（示例）
- 无法通过修改 URL 数字来遍历所有楼盘
- 前端无感知，编码/解码在后端透明处理
