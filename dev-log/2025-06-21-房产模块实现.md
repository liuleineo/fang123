# 2025-06-21 房产模块实现

## 概述
根据 `开发模块/房产模块.md` 的 Spec，在项目中完整实现房产管理模块：
- 后端：Entity/Mapper/Service/Controller 四层，5个管理接口
- 前端：5个 Admin 管理页面
- 数据库：5张新表追加到 schema.sql
- 文档：更新 README.md

## 数据库表（5张）
1. `loupan_tupai_land` — 土拍地块表
2. `loupan` — 楼盘主表（33个字段）
3. `loupan_huxing` — 楼盘户型表
4. `loupan_media` — 媒体素材表
5. `loupan_yfyj` — 一房一价房源表

> 关键适配：Spec SQL 中的 `is_deleted` 统一改为 `deleted`，与项目现有实体一致。
