# 2026-06-28 AI 解析切换至 TokenHub

## 原因
腾讯混元 `hunyuan-turbo` / `hunyuan-turbos-latest` 模型已下线，需要迁移至 TokenHub 平台。

## 变动

### 移除
- `pom.xml` 移除 `tencentcloud-sdk-java-hunyuan` 依赖
- `AiProperties.java` 移除 `hunyuanModel` 字段

### 新增
| 文件 | 说明 |
|------|------|
| `config/TokenHubProperties.java` | TokenHub 配置（apiKey/model/baseUrl） |

### 修改
| 文件 | 说明 |
|------|------|
| `service/AiParseService.java` | Hunyuan SDK → RestTemplate HTTP 调用 TokenHub OpenAI 兼容 API |
| `application-dev.yml` | 新增 `tokenhub` 配置段，移除 `ai.hunyuan-model` |

### 新架构
```
图片 → OCR(SDK) → TokenHub(HTTP OpenAI兼容) → 结构化字段
                         ↑
            POST https://tokenhub.tencentmaas.com/v1/chat/completions
            Authorization: Bearer sk-xxx
            model: hy3-preview
```
