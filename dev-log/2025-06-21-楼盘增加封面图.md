# 2025-06-21 楼盘增加封面图

## 改动

| 层面 | 文件 | 改动 |
|------|------|------|
| 数据库 | ALTER TABLE | `loupan` 新增 `cover_image VARCHAR(500)` |
| schema.sql | db/schema.sql | CREATE TABLE 新增 `cover_image` 字段 |
| 后端 Entity | Loupan.java | 新增 `String coverImage` |
| Admin 管理 | LoupanManage.vue | 基本信息 tab 新增封面上传组件（t-upload），支持上传或手动输入 URL |
| Web 首页 | Home.vue | 卡片封面从占位图标改为显示 `coverImage`，无图时显示图标兜底 |
| Web 详情 | LoupanDetail.vue | 详情页顶部新增封面大图横幅（h-64~h-96），无图时不显示 |

## 效果
- 管理端可上传/填写封面图
- 首页列表卡片优先显示真实封面图
- 详情页顶部展示封面大图
