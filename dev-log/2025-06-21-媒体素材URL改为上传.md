# 2025-06-21 媒体素材URL改为上传

## 改动

### 后端
- `CosService.java`：新增 `uploadFile(MultipartFile, String folder)` 通用文件上传方法，支持图片/视频，最大 20MB
- `LoupanMediaController.java`：新增 `POST /api/admin/medias/upload` 上传接口，返回 `{ url }`

### 前端
- `LoupanMediaManage.vue`：素材URL表单改为 `t-upload` 组件
- 支持拖拽/点击上传图片或视频
- 上传成功后自动填入 URL，也可手动输入
