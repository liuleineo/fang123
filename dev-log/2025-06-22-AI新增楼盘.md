# 2025-06-22 AI新增楼盘功能

## 概述
在楼盘管理页面新增"AI新增楼盘"按钮，上传楼盘资料图片后通过 **腾讯云 OCR + 腾讯混元大模型** 自动识别并解析楼盘信息。

## 技术方案

```
图片上传 → 腾讯云 OCR（文字提取）→ 腾讯混元大模型（语义解析）→ 自动填表
```

**为什么用 AI 解析而不是正则？**
- 楼盘资料格式多样（海报、文档截图、表格），正则只能匹配固定模式
- 混元大模型理解语义，能适配"项目名称"/"楼盘案名"/"项目案名"等多种表述
- 后续维护成本低，无需不断补充正则规则

## 后端改动

### 1. pom.xml - 新增依赖
- `tencentcloud-sdk-java-ocr:3.1.1198` - OCR 通用印刷体识别
- `tencentcloud-sdk-java-hunyuan:3.1.1198` - 混元大模型 ChatCompletions

### 2. application-dev.yml - 新增配置
```yaml
ai:
  secret-id: ...    # 复用 COS 凭证
  secret-key: ...
  region: ap-guangzhou
  hunyuan-model: hunyuan-turbo
```

### 3. 新增文件
| 文件 | 说明 |
|------|------|
| `config/AiProperties.java` | AI 配置属性类（复用 COS SecretId/SecretKey） |
| `dto/AiParseResult.java` | AI 解析结果 DTO，含 imageUrls/ocrText/fields |
| `service/AiParseService.java` | 核心服务：COS上传 → OCR识别 → 混元解析 |

### 4. LoupanController.java - 新增接口
- `POST /api/admin/loupans/ai-parse` - 接收多张图片，返回结构化楼盘字段

## 前端改动

### admin/src/views/LoupanManage.vue
1. "新建楼盘"按钮旁新增 **"AI新增楼盘"** 按钮（橙色轮廓，Sparkles 图标）
2. 新增 AI 对话框（t-dialog）：
   - 图片上传区（最多 5 张，支持 JPG/PNG/WebP）
   - "开始识别"按钮，带 loading 动画
   - 识别结果展示：OCR 原始文本（可折叠）+ 结构化字段列表
   - "确认并填入表单"按钮，一键填入所有识别字段
3. 填入表单后自动打开新建楼盘抽屉，用户可核对修改后保存

## 字段映射
AI 可识别全部 36 个楼盘字段（projectName/district/plate/avgUnitPrice/areaMin/areaMax 等），无法确定的字段自动留空。

## 注意事项
- OCR + AI 调用耗时约 5-15 秒，前端超时设为 120 秒
- 图片先上传 COS 再传给 OCR（ImageUrl 模式，避免 base64 过大）
- 混元大模型返回的 JSON 自动清洗 markdown 代码块标记
- 单张图片限制 5MB，与 COS 上传限制一致
