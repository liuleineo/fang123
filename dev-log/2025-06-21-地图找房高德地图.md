## 功能

- 高德地图展示有经纬度的楼盘标记（蓝色标签）
- 左侧面板：关键词搜索 + 行政区/类型筛选
- 点击列表项 → 地图定位到该楼盘
- 点击地图标记 → 弹出信息窗（封面图+名称+价格+查看详情链接）
- 自适应视野（所有标记完整显示）

## 配置说明

⚠️ 需要高德地图 JS API Key：
1. 前往 https://console.amap.com/ 注册并创建「Web端 JS API」应用
2. 将获取的 Key 填入 `web/src/views/MapSearch.vue` 第 99 行的 `AMAP_KEY` 变量
3. 未配置 Key 时页面显示友好提示

## 楼盘坐标
- 需要在管理端为楼盘填写 `longitude`/`latitude` 后才会在地图上显示
- 当前测试数据「潮映杭园」暂无坐标，需手动填入
