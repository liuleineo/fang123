## 改动
### web/index.html
- 新增高德地图 JS API v2.0 脚本加载
- 配置安全密钥 `_AMapSecurityConfig`

### web/src/views/LoupanDetail.vue
- 头部信息与 Tab 内容区之间新增地图区域（256px/320px 高）
- 根据楼盘的 `longitude`/`latitude` 自动居中定位并打点标记

## 使用说明
1. 前往 [高德开放平台](https://console.amap.com/) 注册并创建应用
2. 获取 **Key**（Web端JS API）和 **安全密钥**（securityJsCode）
3. 替换 `web/index.html` 中的 `YOUR_AMAP_KEY` 和 `YOUR_AMAP_SECURITY_CODE`
4. 重启前端即可
